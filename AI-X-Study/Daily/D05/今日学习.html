🔧 Skill 信息

Skill 名称

PDF（PDF 处理技能）

功能描述

用于读取、提取、合并、分割 PDF 文件内容，支持 OCR 识别扫描件，以及从 PDF 中提取表格和图片。

使用场景

  • 从 PDF 论文中提取文字内容
  • 合并多个 PDF 文件
  • 给 PDF 添加水印
  • 填写 PDF 表单
  • 提取 PDF 中的表格数据

核心参数

参数 说明 示例
source PDF 文件路径 "document.pdf"
pages 指定页面或页面范围 "1-5" / "all"
format 输出格式 "text" / "table"

📸 使用示例

示例 1：读取 PDF 论文内容

描述：从一篇关于 AI 教育的英文论文中提取核心章节内容

命令/操作：

使用 PDF Skill 读取指定论文的第 3-5 页内容

结果：

成功提取论文第 3-5 页内容：

"共生智能（Symbiotic Intelligence）代表了人工智能发展的新范式，强调人机协作而非竞争..."

（提取的文字内容）

示例 2：提取 PDF 表格数据

描述：从课程大纲 PDF 中提取表格形式的课程安排

命令/操作：

使用 PDF Skill 以表格格式提取 PDF 中的课程安排表

结果：
