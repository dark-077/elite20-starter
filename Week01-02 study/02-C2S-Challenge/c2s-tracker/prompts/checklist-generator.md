# AI 质量检查清单生成 Prompt

> 用这个 prompt 让 AI 生成针对 C2S 各产物的质量检查清单

---

## Prompt 模板

```
我正在做 C2S Challenge 的「产物①：论文理解」。

质量最低标准是：看过产物的人能区分哪里是论文讲的、哪里是你讲的。

请生成 7 个检查项，每个检查项要：
1. 能回答「是/否」
2. 能区分「糊弄」和「认真做」
3. 针对这个具体的质量标准

格式：
1. [检查项内容]
   - 糊弄表现：[什么样算糊弄]
   - 认真表现：[什么样算认真]
```

---

## 各产物 Prompt

### 产物①：论文理解

```
我正在做 C2S Challenge 的「产物①：论文理解」。
质量最低标准：看过产物的人能区分哪里是论文讲的、哪里是你讲的。
请生成 7 个检查项，每个检查项要能回答「是/否」，并能区分「糊弄」和「认真做」。
```

### 产物②：KSTAR 行动方案

```
我正在做 C2S Challenge 的「产物②：KSTAR 行动方案」。
质量最低标准：评审教师读完后能说"下学期可以试着跑一下"。
请生成 7 个检查项，每个检查项要能回答「是/否」，并能区分「糊弄」和「认真做」。
```

### 产物③：个人联结

```
我正在做 C2S Challenge 的「产物③：个人联结」。
质量最低标准：必须有具体的"你"——专业、名字、真实生活事实。
请生成 7 个检查项，每个检查项要能回答「是/否」，并能区分「糊弄」和「认真做」。
```

### 产物④：AAR 复盘

```
我正在做 C2S Challenge 的「产物④：AAR 课后复盘」。
质量最低标准：AI 不知道你这 4 天具体发生了什么。
请生成 7 个检查项，每个检查项要能回答「是/否」，并能区分「糊弄」和「认真做」。
```

---

## 使用说明

1. 复制对应产物的 prompt
2. 粘贴到 ChatGPT / Claude 等 AI 工具
3. 根据 AI 输出，创建 Notion 数据库的检查清单
4. 每次提交前，用检查清单自检
