## 🔧 字段类型配置说明

### 文本字段
1. 点击字段标题旁的 **"+"** 按钮
2. 选择 **"文本"**
3. 输入字段名称
4. 按 Enter 确认

### 单选字段
1. 点击字段标题旁的 **"+"** 按钮
2. 选择 **"单选"**
3. 输入字段名称
4. 点击 **"添加选项"** 输入选项名称
5. 按 Enter 确认
6. 重复添加所有选项

### 数字字段
1. 点击字段标题旁的 **"+"** 按钮
2. 选择 **"数字"**
3. 输入字段名称
4. 在格式中选择 **"百分比"**（如需）

### 日期字段
1. 点击字段标题旁的 **"+"** 按钮
2. 选择 **"日期"**
3. 输入字段名称
4. 按 Enter 确认

## ⚡ 快速操作提示

1. **添加新字段**：点击表格顶部最右侧的 **"+"** 按钮
2. **修改字段名称**：双击字段标题
3. **删除字段**：右键点击字段标题 → 选择"删除字段"
4. **调整字段顺序**：拖拽字段标题到新位置
